# Negocio y Trabajo

## Lineas de negocio activas

### 1. Software administrativo homologado
- **PSKloud** (tambien conocido como Premium Soft): sistema de facturacion, contabilidad, nomina, hoteleria, laboratorios, restaurantes, retail. Distribucion bajo licenciamiento en Venezuela.
- **Inbu POS** (Point of Sale): sistema homologado para puntos de venta en Venezuela.

### 2. Servicio de impresoras fiscales
- Centro de servicio autorizado para impresoras fiscales marca **The Factory**.
- Incluye: venta, instalacion, configuracion y soporte tecnico.

### 3. Desarrollo de agentes de IA (linea nueva — prioridad actual)
- Agentes de automatizacion para PyMEs (restaurantes, tiendas, etc.)
- Canal principal actual: **Telegram** (bots de pedidos, delivery, pick-up)
- Flujo: cliente interactua con bot → catalogo → pago (intervencion humana solo en pago)
- Carga masiva de productos via plantillas Excel
- En proceso: pagina web para mostrar y comercializar estos agentes

## Clientes tipicos
PyMEs venezolanas: restaurantes, farmacias, clinicas, hospitales, consultorios medicos y dentales, hoteles, gimnasios, talleres automotrices, peluquerias, spas, pizzerias, tiendas de repuestos, retail en general.
